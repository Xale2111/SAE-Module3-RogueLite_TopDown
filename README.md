Objectifs :
- Commencer avec 3-4 salles.
- On gagne du gold en tuant les méchants
- Une salle bonus peut apparaitre par moment. Elle est soit du type coffre (gold bonus) ou du type arme (meilleure arme)
- lorsqu'on rentre dans une salle, le couloir derrière nous se retire. Lorsqu'on tue tous le monde dans la salle, le/les prochains couloirs se révèlent
- Un (mini) boss est présent dans la dernière salle. Lors'on le tue, on gagne une grosse quantité de gold, un régène de pv et on peut recommencer le donjon avec 1 salle de plus et des ennemis légèrement plus forts.
- un marchand est toujours présent avant le boss. On peut acheter de la vie, des PVs max en plus, une meilleure arme (double boost), un boost de dégat permanent, un boost de gold permanent. (seul 2 options sur 5 seront dispos)
